localStorage.key()

key() 作为 Storage 接口的方法，接受一个数值 n 作为参数，返回存储对象第 n 个数据项的键名。

            localStorage.setItem('key1', 'value1');
            localStorage.setItem('key2', 'value2');
            localStorage.setItem('key3', 'value3');

            localStorage.key(0);
            localStorage.key(1);
key()方法 getItem('key1') key(0) key(1)
期望值 value1 key1 key2
