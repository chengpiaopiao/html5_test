localStorage

检测支持 描述
supports 检测浏览器是否支持 Storage
属性 描述
length length 是 Storage 接口的只读属性，返回一个整数，表示存储在 Storage 对象里的数据项（data items）数量。
方法 描述
clear() 清空存储对象里所有的键值。
getItem() 接受一个键名（key name）作为参数，并返回对应键名的值（key's value）。
setItem() 接受一个键名和值作为参数，将会把键名添加到存储中，如果键名已存在，则更新其对应的值。
removeItem() 接受一个键名作为参数，会把该键名从存储中移除。
key() 接受一个数值 n 作为参数，返回存储对象第 n 个数据项的键名。
事件 描述
event storage 事件主要是用于监听 localStorage 数据改变时，通知其他窗口或者标签
